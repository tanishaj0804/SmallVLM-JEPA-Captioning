# Temporal-Aware Video Captioning using SmolVLM2 and V-JEPA 2

## Overview

Traditional Vision-Language Models (VLMs) are good at understanding the visual content of a video. They can identify people, objects, environments, and describe the overall scene.

However, visually similar videos may contain very different actions.

For example, a person **picking up a cup** and a person **putting down a cup** may contain the same person, object, and background. A scene-focused VLM may generate similar descriptions for both videos while missing the difference in motion.

This project combines **visual scene understanding** and **temporal action recognition** to generate more motion-aware video captions.

---

## Project Idea

The system uses three models:

### SmolVLM2 — Scene Understanding

SmolVLM2 analyzes the video and generates a detailed description of the visible scene.

It identifies information such as:

- The environment or setting
- People present in the scene
- Important objects
- Object appearance and attributes
- Spatial relationships between objects

Example:

> A person is standing near a table and holding a transparent bottle beside a drinking glass.

---

### V-JEPA 2 — Temporal Action Understanding

V-JEPA 2 analyzes the motion occurring throughout the video.

The V-JEPA 2 model used in this project contains an action classification probe trained on the Something-Something V2 action dataset.

Instead of generating sentences, it predicts action labels.

Example:

> Pouring [something] into [something]

This provides temporal information about **what is happening over time**.

---

### TinyLlama — Caption Fusion

The scene description generated by SmolVLM2 and the action predicted by V-JEPA 2 are passed to TinyLlama.

TinyLlama combines both sources of information into one natural and factual video caption.

Example:

**SmolVLM2**

> A person is standing near a table holding a transparent bottle beside a drinking glass.

**V-JEPA 2**

> Pouring [something] into [something]

**Final Caption**

> A person standing near a table pours liquid from a transparent bottle into a drinking glass.

## System Pipeline

```text
                    Input Video
                         │
            ┌────────────┴────────────┐
            │                         │
            ▼                         ▼
       SmolVLM2                   V-JEPA 2
   Scene Understanding         Action Recognition
            │                         │
            ▼                         ▼
     Scene Description           Action Label
            │                         │
            └────────────┬────────────┘
                         │
                         ▼
                     TinyLlama
                   Caption Fusion
                         │
                         ▼
              Temporal-Aware Caption

Project Structure
SmallVLM+JEPA Captioning/
│
├── scene_caption.py
├── action_probe.py
├── fuse_tinyllama.py
├── pipeline.py
│
├── requirements.txt
└── README.md
```

_scene_caption.py_

Uses SmolVLM2 to analyze the visual scene and generate a scene description.


_action_probe.py_

Uses the V-JEPA 2 action classification probe to predict the temporal action occurring in the video.


_fuse_tinyllama.py_
Uses TinyLlama to combine the scene description and detected action into a single natural caption.


_pipeline.py_

Runs the complete video captioning pipeline.

**Running the Project**

Install the required dependencies:

      pip install -r requirements.txt

Run the complete pipeline:

      python pipeline.py videos/v1.mp4

Each model can also be tested independently:

      python scene_caption.py videos/v1.mp4
      python action_probe.py videos/v1.mp4
      python fuse_tinyllama.py --scene "A person is standing near a table." --action "Picking [something] up"

**Goal**

The goal of this project is to improve video captioning by combining:

Semantic scene understanding from Vision-Language Models
Temporal motion understanding from video representation models

By combining SmolVLM2 and V-JEPA 2, the system aims to distinguish between visually similar videos containing different actions and generate captions that better represent what is actually happening over time.
